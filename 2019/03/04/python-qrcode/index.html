介绍 qrcode

QrCode 是一个二维码生成器，可生成普通二维码、带图片的艺术二维码（黑白与彩色）、动态二维码（黑白与彩色）。

安装qrcode 需要的依赖库有 pillow, numpy, imageio

-n (name) 定义名字 支持的文件格式有 .jpg, .png, .bmp, .gif
-d (directory) 定义位置

参数 -p 用来将QR二维码图像与一张同目录下的图片相结合，产生一张黑白图片。

加上参数 -c 可以使产生的图片由黑白变为彩色的。

生成动图的注意事项

生成动图的方式和普通的生成二维码方式一样，但名字必须保持 .gif 格式。

注意事项

  1. 最好采用正方形图片

可用字符

  1. 0-9
  2. 大小写英文字母
  3. 常用英文标点和空格
