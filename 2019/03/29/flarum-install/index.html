安装 Flarum 论坛详细过程

Posted on 2019-03-29 | In 杂货 | Comments:

Flarum， 是一款界面简约好看的论坛软件，用php编写。
使用 flarum 需要 LNMP， 以及 composer

安装 composer

使用 Ubuntu 系统, Ubuntu 18.04 配置 LNMP 点这里

其他系统

安装 flarum

安装 flarum 之前需要安装 php 的一些扩展 dom, gd, json, mbstring, openssl, pdo_mysql, tokenizer
还需要安装一个 ext-curl 扩展，这个扩展在 composer 安装时尤为重要。

在你想运行 flarum 的文件夹下，注意该文件夹为空文件夹

安装成功以后，会生成一些文件，注意如果没有 vendor 文件夹。则说明安装失败，你可以在 flarum 官网询问解决办法。

运行 flarum

配置 nginx 文件
在 /etc/nginx/sites-available 文件夹下，具体可以看该网站以前的文章。
将 root 指向 你安装 flarum 文件的 public 文件夹，删除 {location / {} }所有内容，加上 flarum 为我们写的，在 {server{}} 中加上 include flarum 文件夹/.nginx.conf,注意该文件为隐藏文件。
