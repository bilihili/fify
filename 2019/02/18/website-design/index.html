如何同步代码

感觉用ftp传输不太现实，所以我现在的办法是用 git 同步代码，现在 GitHub 可能是被微软收购的原因，允许非付费用户使用 private 仓库，这样对我也是特别方便，但是我对 Git 还不是特别熟悉，所以遇到了一些问题。
因为项目使用的是 django 所以数据库暂时用 sqlite3 ,因为数据库在开发时和网站上都可能被修改，所以我打算设置 git 的忽略文件。

需要保存默认的avatar以及migrations文件,其他图片不保存，顺便吐槽一下千牛，他的存储方案需要备案的域名。亚马逊的aws s3可以不用，但是昨天试了一试发现不能上传文件，而且告知我没有修改它的权限，可能是我不会设置的原因。
接下来是 Git 的原因，当我在服务器是pull时他告知我，我修改了服务器上的文件，需要我自己手动 merge，merge 是不可能 merge 的，所以我用了这个命令。
