关于 nginx 的文件结构

  • /etc/nginx
    这个是 nginx 的根文件目录
  • /etc/nginx/nginx.conf
    设置 nginx 的全局服务
  • /etc/nginx/conf.d/
    默认的 hhtp 服务器配置文件目录

    nginx 命令

  • nginx -h
    帮助目录
  • nginx -v
    nginx 版本
  • nginx -V
    展示更加详细的 nginx 相关信息
  • nginx -t
    测试 nginx 的配置
  • nginx -T
    测试 nginx 配置文件，并获取相关帮助信息
  • nginx - s signal
    -s 是 send 的缩写 signal 可以用 stop, reload, quit, reopen 代替
    stop 立即停止 nginx， quit 待相关服务停止后再关闭，reload 代表更新配置文件。
