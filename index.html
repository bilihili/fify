使用 flarum 需要 LNMP， 以及 composer

安装 composer

使用 Ubuntu 系统, Ubuntu 18.04 配置 LNMP 点这里

其他系统

安装 flarum

安装 flarum 之前需要安装 php 的一些扩展 dom, gd, json, mbstring, openssl, pdo_mysql, tokenizer
还需要安装一个 ext-curl 扩展，这个扩展在 composer 安装时尤为重要。

在你想运行 flarum 的文件夹下，注意该文件夹为空文件夹

Posted on 2019-03-23 | In c++ | Comments:

当我们在进行数组排序时，我们想到的是写一个函数，传递数组进去，让它返回一个排序之后的数组出来，像JAVA,Python,等语言都有返回数组的功能，而C 语言没有返回整个数组的功能，然而我们可以返回一个指向数组的指针。这里就显得指针在 C 语言的重要之处。

用这种办法就可以方便对数组进行排序。

注意声明数组 array 后，将array的指针传给了 sortArray, 而不是将数组传递给 sortArray， 通过 *(array+i) 获取数组 array[i] 。

创建 EC2 实例

要开始使用 EC2，您需要启动一个被称为 Amazon EC2 实例的虚拟服务器。我的地区选在 Ohio ,你可以在菜单栏中修改地区。

选择一个实例对象

选择 t2.micro ,这个时对应的免费一年的版本。

设置存储
