注册条件

拥有一张信用卡，VISA，或者Mastercard

创建 EC2 实例

要开始使用 EC2，您需要启动一个被称为 Amazon EC2 实例的虚拟服务器。我的地区选在 Ohio ,你可以在菜单栏中修改地区。

选择 Amazon 系统镜像

我选择的是 Ubuntu 18.04 ,尽量避免选择 Red Hat Enterprise Linux 7.6 。

选择一个实例对象

选择 t2.micro ,这个时对应的免费一年的版本。

设置存储

免费版本最大的存储空间为 30 G 。

配置安全组

这里添加 http 80 端口，https 443 端口。

配置密钥对

成功以后会有一个 name.pem 文件，保存下来，名字可以任意取。

ssh 连接 EC2

在 .pem 文件目录下，打开 Git Bash

公有 DNS 在运行实例的状态栏下面可以看到。

另外也有一个公有 IP 地址，每月流入流量不计，流出免费流量为 15G.
